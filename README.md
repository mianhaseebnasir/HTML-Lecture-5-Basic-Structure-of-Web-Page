<h1>HTML Lecture 5 – Basic Structure of Web Page</h1>

<h2>📖 Description</h2>
This project demonstrates the basic structure of an HTML webpage. It includes heading, paragraph, and script tag implementation.<br>
This assignment was given in my class lecture to understand fundamental HTML concepts.
<hr>

<h2>📂 Project Overview</h2>
<h4>🔹 Features</h4>
<Li>Basic HTML5 document structure</Li>
<Li>Usage of <head> and <body> tags</Li>
<Li>Heading tag implementation (h1)</Li>
<Li>Paragraph tag implementation (p)</Li>
<Li>External JavaScript file linking using (script src="") </Li>
<hr>

<h2>🛠️ Technologies Used</h2>
<li>HTML5</li>
<hr>

<h2>🎯 Learning Purpose</h2>
This project was created to understand:
<Li>How to structure a simple HTML document</Li>
